Kobe Bryant Tribute Page | 傳奇致敬網頁

專案簡介
這是一個使用 HTML5, CSS3 與 原生 JavaScript (Vanilla JS) 開發的單頁式響應式網頁。本專案不僅是我練習前端開發基礎的成果，更是為了向傳奇球星 Kobe Bryant 表達緬懷與敬意。

技術核心
視覺設計：使用 CSS3 實作 視差滾動 (Parallax Scrolling) 效果，並搭配半透明磨砂玻璃質感 (Glassmorphism) 的內容區塊，解決背景圖對文字閱讀的干擾。

動態互動：撰寫原生 JavaScript 監聽捲軸事件 (Scroll Event)，實作 Scroll Reveal 動畫，讓內容區塊在滑入視野時優雅漸顯。

標準化優化：利用 CSS @keyframes 動畫取代已過時的 <marquee> 標籤，確保網頁符合當前 Web 規格。

多媒體整合：嵌入 YouTube Iframe API，並透過參數設定精確的播放起始點。

專案亮點
具備現代化的 UI/UX 視覺層次感。

程式碼模組化，邏輯清晰易於維護。

完全響應式設計，確保流暢的閱讀體驗。


Project Overview
A single-page responsive website developed using HTML5, CSS3, and Vanilla JavaScript. This project serves as a showcase of my foundational front-end development skills and a personal tribute to the legendary Kobe Bryant.

Key Tech Stack
Visual Design: Implemented Parallax Scrolling and Glassmorphism using CSS3 to ensure high-quality aesthetics and content readability against complex backgrounds.

Interactive Logic: Developed a Scroll Reveal system using Vanilla JS to trigger fade-in animations as users scroll through different sections.

Standardization: Used CSS @keyframes for smooth text scrolling animations, adhering to modern web standards by replacing deprecated tags.

Media Integration: Embedded YouTube Iframe API with timestamp parameters for precise video content delivery.

Highlights
Modern and professional UI/UX design with strong visual hierarchy.

Clean, modular code structure following best practices.

Smooth interactive experience with optimized asset loading.
